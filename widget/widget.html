Web-RAG Widget Test Page

The chat widget should appear as a blue bubble in the bottom-right corner.

Embedding code:
<script src="http://localhost:8090/widget.js" data-site-id="2" data-title="MSU Assistant"></script>

Customization attributes:
• data-site-id — which indexed site to query
• data-api — API server URL (default: same origin)
• data-title — chat panel header text
• data-color — accent color (hex)
• data-position — "right" or "left"
